## Solidity 

#### Three type of Comment 
- single-line comments (`//`)
- multi-line comments (`/* ... */`)
- `///` or `/** ...*/`

#### Import
 In solidity, you can reference other solidity source files so you can use them in your current source files. 
```
import "./token.sol";
import "./Token.sol" as token;
```

- Address
